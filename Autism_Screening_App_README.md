# 🧠 Early Autism Screening App

An AI-powered screening application designed to assist in the early
detection of autism spectrum indicators in infants and young children.

------------------------------------------------------------------------

## 🚀 Overview

Early detection of autism can significantly improve developmental
outcomes. This application provides a guided screening tool for parents
and caregivers to identify early behavioral signs and risk indicators.

⚠️ **Disclaimer:** This app is NOT a diagnostic tool. It is intended for
early screening and awareness only. Always consult a medical
professional for diagnosis.

------------------------------------------------------------------------

## ✨ Features

### 👶 Screening Questionnaire

-   Age-appropriate behavioral questions
-   Based on early developmental milestones
-   Simple Yes/No responses

### 📊 Risk Assessment

-   Intelligent scoring system
-   Categorizes results into:
    -   Low Risk
    -   Moderate Risk
    -   High Risk

### 🎯 User-Friendly UI

-   Clean medical-themed interface
-   Easy navigation between questions
-   Mobile-friendly design

### 📄 Result Summary

-   Clear explanation of results
-   Guidance on next steps
-   Recommendations for professional consultation

------------------------------------------------------------------------

## 🧠 How It Works

1.  User selects child's age group
2.  Answers screening questions
3.  System calculates score
4.  Displays risk level with explanation

------------------------------------------------------------------------

## 🏗 Architecture

### 🔹 Frontend

-   Built using modern UI frameworks (Streamlit / React)
-   Handles:
    -   Question display
    -   User input
    -   Result visualization

### 🔹 Backend

-   Handles:
    -   Scoring logic
    -   Data processing
    -   Optional storage (if enabled)

------------------------------------------------------------------------

## ⚙️ Screening Logic

-   Each response contributes to a risk score
-   Higher scores indicate higher likelihood of developmental concerns
-   Threshold-based classification:
    -   Low Risk → Continue monitoring
    -   Moderate Risk → Observe closely
    -   High Risk → Seek professional evaluation
